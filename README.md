## 2. Base(0.0), Negativo (256.0), Gamma(0.256), Logaritmo(256,256)

![image](https://github.com/user-attachments/assets/03674e34-82c0-4e82-adb0-e1bc1491d4d5)

**Negativo:**  
`s = 255 - r`

**Gamma:**  
`s = (1 / 255^{γ-1}) * r^γ`

**Logaritmo:**  
`s = (256 / ln(256)) * ln(1 + r)`
